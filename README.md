# My Blog API

Это REST API для блога, написанное на **Django** и **DRF**. 

## Функционал
* Регистрация и авторизация пользователей (Token Auth).
* Создание постов и комментариев.
* Архитектурно правильное создание комментариев через `/posts/{id}/comments/`.
* Права доступа: редактировать может только владелец.

## Как запустить
1. Клонировать репозиторий.
2. Создать и активировать venv: `python -m venv venv`.
3. Установить зависимости: `pip install -r requirements.txt`.
4. Запустить миграции: `python manage.py migrate`.
5. Запустить сервер: `python manage.py runserver`.

## Документация
Доступна через Swagger по адресу: `/swagger/`
